# CyberSecuirtyAwarenessBot
 Project Overview
This is a C# console-based chatbot developed for cybersecurity awareness. 
It helps users learn about password safety, phishing, suspicious links, and safe browsing.

## Features
- Voice greeting using WAV audio
- ASCII art title screen
- Personalised user interaction
- Basic cybersecurity responses
- Input validation
- Coloured console interface
- Typing effect
- Structured code using classes and methods
- GitHub Actions CI workflow

## How to Run
1. Open the project in Visual Studio.
2. Ensure the file Assets/greeting.wav exists.
3. Build and run the application.

## Example Questions
- How are you?
- What is your purpose?
- What can I ask you about?
- Tell me about password safety
- What is phishing?
- How do I browse safely?
